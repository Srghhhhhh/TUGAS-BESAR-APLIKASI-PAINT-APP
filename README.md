# Deskripsi Aplikasi

Paint App merupakan aplikasi menggambar sederhana yang memungkinkan user untuk membuat gambar digital secara mudah dan interaktif. Pada aplikasi ini, user dapat menggambar menggunakan berbagai pilihan warna dan ukuran brush sesuai kebutuhan.

Selain fitur menggambar bebas, aplikasi juga menyediakan beberapa tools seperti penghapus (eraser), clear canvas untuk menghapus seluruh gambar, serta penyimpanan hasil gambar. Interface aplikasi dirancang sederhana agar mudah digunakan oleh semua kalangan pengguna.
